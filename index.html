Chibi Engine

A modern and powerful TypeScript 2D game engine (WIP)
This project is still in early development and not yet ready for use.
Chibi Engine aims to offer a modern and simple approach to game development using web technologies. Inspired by popular game engines, it offers a modern and easy to use API, taking advantage of TypeScript and ES6 features such as asynchrony, classes and mixins. This encourages a maintainable, flexible and extensible programming methodology while providing users with the common features to create games.
